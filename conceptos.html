La consola de Javascript

Muestra información sobre la página web que se esta ejecutando en ese momento, y tambien incluye la línea de comando que puedes usar para ejecutar expresiones Javascript() en la página actual.

Comentarios

Javascript tiene dos tipos de comentarios:



Los comentarios son anotaciones en el código fuentes de un programa que son ignoradas por el interprete.
Los comentarios deben usarse para describior aspectos importantes, por ejemplo aspectos que permitan una mejor comprensión del código.
