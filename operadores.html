Operadores de comparación

Igualdad: Devuelve true si ambos operandos son iguales (==).
Desigualdad: Devuelve true si ambos operandos no son iguales (!=).
Estrictamente iguales: Devuleve true si ambos operadores son iguales y tienen el mismo tipo (===).
Estrictamente desiguales: Devuleve true si los operandos no son iguales y/o no son del mismo tipo (!==).
Mayor que: Devuelve true si el de la izquierda es mayor (>).
Mayor o igual que: Devuelve true si el de la izquierda es mayor o igual (>=).
Menor que: Devuelve true si el de la izquierda es menor (<).
Menor o igual que: Devuelve true si el de la izquierda es menor o igual (<=).


Operadores Aritmeticos

Además de las operaciones aritmeticá estandar (+, -, *, /). Javascript brinda los siguientes operadores aritmeticos:



Resto (%): Corresponde al módulo de una creación. Devuelve el resto de la división.

Ejemplo:
12%5 devuelve 2.

Incremento (++): Incrementa en una unidad al opereando. Si es usado: ++x duvuelve el valor del operando después de añadirle 1. Si se usa =x++ devuelve el valor antes de añadirle 1.

Ejemplo:
Si x=3
++x dara x=4
x++ dara 3 y luego establece x=4.

Decremento(--): Resta una unidad al operando. Dependiendo la posición con respecto al operando, tiene el mismo comportamiento que el incremento.

Ejemplo:
Si x=3
--x dara x=2
x-- dara 3 y luego establece x=2.


Operador de concatenación

Une dos valores de tipo string, devolviendo otro string correspondiente a la unión.

Ejemplo:
console.log("mi" + "string"); // mi string


Operador ternario

Necesita 3 operandos:
condición? valor 1: valor 2
Si la condición es true tomara el valor 1, sino el valor 2.
